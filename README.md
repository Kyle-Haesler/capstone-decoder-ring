# Decoder Ring


## Project Description
Application that will encode and decode different messages based on different cyphers.

## Instructions
- Download files
- Run npm install
- Run npm start to run the project locally
## Skills Demonstrated
- Coding complex JavaScript functions
- Writing unit tests with Mocha and Chai
## Technologies
- JavaScript ES6
- HTML5
- Node.js
- Bootstrap
- Mocha
- Chai
